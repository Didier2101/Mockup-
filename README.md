# Yo Sé Quien Sabe - Plataforma de Asesoría

Mockup responsive de la plataforma "Yo Sé Quien Sabe" que conecta asesorados con expertos.

## Características Implementadas

### Vista 1 - Pantalla de Inicio
- Eslogan "YO SE QUIEN SABE LO QUE USTED NO SABE"
- Selección de tipo de usuario (Asesorado/Experto)
- Términos y condiciones
- Política de tratamiento de datos

### Vista 2 - Registro
- **Campos para ambos tipos:**
  - Tipo de identificación (Persona Natural/Jurídica)
  - Número de identificación
  - Sector según DANE (Agropecuario, Industrial, Construcción, Comercio, Servicios)
  - Tamaño de empresa (Micro, Pequeña, Mediana, Grande)
  - Habeas Data

- **Campos específicos para Expertos:**
  - Perfil de LinkedIn
  - Temas de expertise
  - Años de experiencia
  - Disponibilidad (Agenda/En línea)

### Vista 3 - Solicitud
- Selección de tema (lista desplegable completa)
- Duración de sesión (30, 60, 90 minutos)
- Descripción detallada de la consulta

### Vista 4 - Selección de Experto
- Perfiles de expertos con:
  - Foto/avatar
  - Calificación
  - Años de experiencia
  - LinkedIn verificado
  - Precios por duración
  - Estado de disponibilidad
- Botones para selección, llamada inmediata y chat

### Vista 5 - Pago
- Métodos de pago: PSE, Nequi, DaviPlata
- Información para facturación
- Dispersión de fondos

### Vista 6 - Sesión Programada
- Resumen de la sesión
- Concepto generado por IA
- Recordatorios automáticos
- Botones para unirse a videollamada/chat

### Vista 7 - Perfil de Experto
- Información completa del experto
- Precios por duración
- Estado en línea/offline
- Gestión de agenda
- Consultas pendientes

## Tecnologías Utilizadas
- HTML5
- CSS3 (Flexbox, Grid, Variables CSS)
- JavaScript (ES6)
- Font Awesome para iconos
- Diseño completamente responsive

## Archivos del Proyecto
- `index.html` - Estructura principal
- `styles.css` - Estilos y diseño responsive
- `script.js` - Funcionalidad e interacciones
- `README.md` - Documentación

## Instalación y Uso
1. Clonar el repositorio
2. Abrir `index.html` en un navegador web
3. Navegar entre las diferentes vistas usando el panel lateral

## Despliegue
Este proyecto está listo para desplegar en Netlify, GitHub Pages o cualquier servicio de hosting estático.
